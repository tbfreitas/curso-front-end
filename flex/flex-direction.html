A propriedade flex-direction especifica como os flex-items são organizados no container dizendo qual a direção do eixo principal do container.

Quando a opção *-reverse é utilizada, note que apenas os ITENS são colocados de maneira reversa, isso não afeta a direção de escrita, imagens ou formatação de texto interna.

O valor padrão sempre será row.

row

O eixo principal será o mesmo de elementos inline e direção de escrita, que para nós é da esquerda para a direita.

row-reverse

O mesmo que row, porém os itens se iniciam da direita para a esquerda dentro do container.

column

O eixo principal passa a ser o mesmo de elementos block, e também segue nosso padrão de escrita de cima para baixo.

column-reverse

O mesmo de column exceto que se inicia de baixo para cima no container.
